Erstellen Sie eine Kundenzufriedenheitsumfrage und binden diese per Shortcode im Frontend ein.

== Description ==

Mit dem Plugin k&ouml;nnen Sie im Backend einen Fragebogen mit individuellen Fragen definieren und im Frontend f&uuml;r Ihre Besucherinnen und Besucher per Shortcode einbinden.

Fragearten:

*   Freitext-Anwort
*   Ja/Nein-Antwort
*   Bewertung auf einer 5-stufigen Bewertungsskala

Funktionen:

*	Legen Sie bis zu 10 Fragen f&uuml;r Ihre Kundenbefragung an
* 	Binden Sie die Umfrage per Shortcode im Frontend ein
* 	Integrierter Spamschutz (Sicherheitsabfrage / Honeypot)
* 	Die gesendeten Antworten sind im Backend einsehbar

== Installation ==

1. Laden Sie das Plugin in das Verzeichnis `/wp-content/plugins/`
2. Aktivieren Sie das Plugin in Ihrem Backend
3. Binden Sie den Shortcode [rp_kundenbewertung] auf Ihrer Seite ein.
